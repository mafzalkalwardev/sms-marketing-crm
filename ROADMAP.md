# SignalMint Roadmap

## Now

- Business texting workspace as the default product experience.
- Messages-first navigation.
- Contacts, Dialpad / New Text, Numbers, and Settings.
- Mock SMS mode for local testing.
- Vonage SMS support when credentials are configured.
- STOP/unsubscribe handling.
- Demo seed data.

## Next

- Live inbox refresh with WebSockets or polling.
- Better contact notes and tags editing from the conversation sidebar.
- Team assignment and shared inbox ownership.
- Conversation close/archive states.
- Real call controls if a voice provider is added.

## Later

- Broadcasts as a secondary feature, not the core product.
- Redis/BullMQ queue for high-volume sends.
- Billing/subscriptions.
- Organization/team administration.
- Production deployment with HTTPS, backups, monitoring, and provider webhook verification.
